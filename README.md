**통계 데이터분석**
* 기술통계
  - 상관분석, 회귀분석(단순,다중): 결정(회귀)계수, P-Value<0.05, 잔차 합 = 0
* 추론통계
  - 단측검정, 양측검정
  - T-test(T검정) : 모집단의 분산을 모를 때 표본의 분산/표준편차를 통해 두 모집단의 평균을 추정 비교하는 검정
    + 정규화 : 화폐가치 등 기준이 다 다를 때 0-1 값으로 변환
